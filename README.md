# Netflix Movies & TV Shows Data Analysis

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python to understand content distribution and trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
- Excel

## Key Steps
- Handled missing values
- Converted date columns and extracted year
- Analyzed shows by country, genre, and year
- Visualized insights using charts
- Exported cleaned data to Excel

## Dataset
Netflix Movies and TV Shows dataset from Kaggle
